# File Indexer

A Python file indexer project with core components for traversing directories, extracting file metadata, indexing, and storage.

## Project Structure

```
file-indexer/
├── core/
│   ├── __init__.py
│   ├── traverser.py
│   ├── extractor.py
│   ├── indexer.py
│   └── storage.py
├── tests/
│   └── __init__.py
├── pyproject.toml
└── README.md
```

## Setup

```bash
poetry install
```

## Usage

```python
from core.traverser import Traverser
# ...
```
